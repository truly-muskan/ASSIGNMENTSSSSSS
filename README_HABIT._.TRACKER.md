Smart Habit Tracker

Description

Smart Habit Tracker is a Python-based application that helps users track their daily habits and monitor progress.

Features

- Add a habit
- View habits
- Mark habits as completed
- Delete habits
- Track completion percentage
- Motivational feedback messages

Concepts Used

- Variables and Data Types
- Operators and Expressions
- Control Flow
- Loops
- Functions
- Object-Oriented Programming (OOP)

How to Run

python SMART_HABIT._.TRACKER.py
